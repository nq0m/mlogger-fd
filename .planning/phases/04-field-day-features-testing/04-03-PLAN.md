    4. **loadSound(name)** — Checks `buffers[name]` cache. If missing: fetches `/audio/${name}.wav` (per D-06, audio files served from `frontend/static/audio/` via embed.FS), gets arrayBuffer, calls `audioCtx.decodeAudioData(arrayBuffer)`, caches in `buffers[name]`, returns buffer. Error in fetch/decode → throws (caught by playSound()).

    5. **export async function playSound(name)** — Guards: if `audioState.muted` return. Calls `ensureContext()`. Wraps in try/catch: loads sound, creates `audioCtx.createBufferSource()`, sets `source.buffer`, connects to `audioCtx.destination`, calls `source.start(0)`. On error, `console.warn('Audio playback failed:', e)` — never throws to caller.
