<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→API | Untrusted JSON body crosses from browser to Go handler via POST /api/qso |
| client→API | URL query parameters cross via GET /api/qso?limit=&offset= |
| server→filesystem | Go binary reads/writes SQLite database file |
| server→embedded FS | embed.FS serves pre-compiled static files (read-only) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | POST /api/qso handler | mitigate | Use parameterized queries (? placeholders) for all SQL operations via database/sql; never string-concatenate user input into SQL |
| T-01-02 | Information Disclosure | XSS via QSO data in LogTable | mitigate | Svelte auto-escapes `{expression}` output; never use `{@html}` for user-provided callsign/exchange data |
| T-01-03 | Information Disclosure | Path traversal via SPA file serving | mitigate | embed.FS is compile-time read-only; only files known at build time are embedded; http.FileServer sanitizes URL paths |
| T-01-04 | Tampering | JSON injection in API responses | mitigate | encoding/json properly escapes all strings; all responses use json.NewEncoder(w).Encode(struct) |
| T-01-05 | Tampering | Mass assignment via JSON body on POST /api/qso | mitigate | Decode into explicit struct (model.CreateQSOInput) with only expected fields; never decode into map[string]interface{} and pass to SQL |
| T-01-06 | Tampering | SQL injection via GET /api/qso query params | mitigate | Parse limit/offset as integers with strconv.Atoi; use ? placeholders for LIMIT/OFFSET values |
| T-01-07 | Elevation of Privilege | No authentication on API endpoints | accept | LAN-only trusted environment per PROJECT.md; all operators in tent are trusted; auth deferred to Phase 2 if open WiFi needed |
| T-01-SC | Tampering | npm install / go get package supply chain | mitigate | Blocking human checkpoint (Task 1) before any package installation; all packages reviewed against npmjs.com and pkg.go.dev |
</threat_model>
