  <files>internal/qso/dupe.go, internal/qso/dupe_test.go, internal/handler/dupe.go</files>
  <behavior>
    Exact dupe check: given callsign="K1ABC", band="20M", mode="CW" where a prior QSO exists with same callsign+band+mode and is_dupe=0, CheckDupe returns isDupe=true.
    Non-dupe: given callsign="K1ABC", band="20M", mode="SSB" where prior QSO exists on 20M CW only (different mode), CheckDupe returns isDupe=false.
    Partial similarity: given callsign="K1X" where "K1XX" exists in log, CheckSimilarCall returns ["K1XX"]. Given callsign="W1AW" with no similar calls, returns empty slice.
    Empty database: any callsign returns isDupe=false, similarCalls=[].
    Validation: ValidateCallsign("") → warning "Callsign required". ValidateCallsign("K") → warning "Callsign too short". ValidateCallsign("K1ABC") → no warning.
  </behavior>
